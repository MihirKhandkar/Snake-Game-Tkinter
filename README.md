# Snake Game (Tkinter - Python)

A classic Snake Game built using Python and Tkinter.
This project focuses on GUI programming, event handling, and game logic implementation from scratch.

---

## 🚀 Features

* Smooth snake movement using `after()` game loop
* Keyboard controls (arrow keys)
* Food generation with grid alignment
* Score tracking system
* Snake growth on eating food
* Wall collision detection (Game Over)
* Self-collision detection
* Centered game window for better UX

---

## 🎮 Controls

* ⬆️ Up Arrow → Move Up
* ⬇️ Down Arrow → Move Down
* ⬅️ Left Arrow → Move Left
* ➡️ Right Arrow → Move Right

---

## 🛠️ Tech Stack

* **Python**
* **Tkinter (GUI Library)**
* **Random module**

---

## 🧠 Concepts Used

* Canvas drawing (`create_rectangle`, `create_oval`)
* Event handling (`bind`, `event.keysym`)
* Game loop using `after()`
* List-based snake representation
* Collision detection (walls & self)
* Dynamic UI updates (`label.config`)
